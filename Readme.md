# CT Lib

CT Lib is a C++ library that can read and write Nintendo file formats that are
used by Mario Kart Wii. It is licensed under GPLv3+.

## Format Support

| **Format**      | **Status**         |
| --------------- |:------------------:|
| Yaz0/Yaz1       | Fully Supported    |
| U8              | Fully Supported    |
| BRRES           | In Progress        |
| KCL             | Not Supported Yet  |
| KMP             | Not Supported Yet  |
